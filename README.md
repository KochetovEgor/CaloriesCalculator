# Калькулятор калорий

Это приложение позволяет пользователю указывать свой дневной рацион и считать потребление калорий, белков, жиров и углеводов (КБЖУ).

## Эндпоинты

Backend приложения реализован в виде HTTP API. Далее описаны эндпоинты HTTP контроллера.

### /login

Позволяет авторизоваться в приложении. Возвращает JWT, который необходимо прикладывать почти ко всем другим эндпоинтам.

### /register

Позволяет пользователю зарегистрироваться в приложении.

### /product/add

Позволяет пользователю добавить какую-либу еду из его рациона, указав её КБЖУ.
